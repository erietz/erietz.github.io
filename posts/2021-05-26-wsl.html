Fix WSL File Permissions

The file permissions on Windows Subsystem for Linux are very screwed up by default. To fix this, create the file /etc/wsl.conf if it does not already exist and add the following:

[automount]
enabled = true
options = "metadata,umask=22,fmask=11"
